# diceroll.vim
__Need to check *Sanity values* after seeing a huge number of error logs? diceroll.vim can help.__

## Instration
Packer
```
use 'Allianaab2m/diceroll.vim'
```

vim-plug
```
Plug 'Allianaab2m/diceroll.vim'
```

## Usage
Just Type,
```
:Diceroll [times] [dice]
```

example:
```
:Diceroll 1 100 <-- 1d100
:Diceroll 3 6   <-- 3d6
```
